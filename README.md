## Directory guide

folder | contents
---|---------
`code` |  R scripts
`credentials` | API credentials
`data` | raw data files
`markdown` | markdown files documenting analyses/creating reports
`output` |  files (e.g., .pngs) created by analyses
`rdata` | .RData files
